Grid

The Gaming Docs digital manual reader has a responsive layout based on the Foundation 12-column nestable grid, using Media Queries to switch between various layouts. The following breakpoints determine each layout.

  
    $mobile: "only screen and (min-width:'#{$mobile-screen}')";
    $small: "only screen and (min-width:'#{$small-screen}')";
    $medium: "only screen and (min-width:#{$medium-screen})";
    $large: "only screen and (min-width:'#{$large-screen}')";

Layout

There is a default stylesheet for the base template as well as theme overrides specific to individual manuals. Stylesheets are written in SCSS, broken down into partials and compiled into a master CSS file.

There are three main sections to the reader layout: Top Bar, Content Area and Table of Contents.

Top Bar

The top bar contains the manual title and charm icons for Search, Table of Contents (TOC) and Settings. Charms are only visible on larger screen resolutions and are hidden for tablet and mobile where a standard mobile sidebar toggle is displayed instead.

The top bar has a fixed height of 50px and a linear background CSS3 gradient from left to right with $dark-blue to $medium-blue color-stops.
